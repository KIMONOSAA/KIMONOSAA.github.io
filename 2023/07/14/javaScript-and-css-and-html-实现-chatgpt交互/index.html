项目大总结

辛苦了一下午终于写完啦，本小项目用了fetch API来进行跟gpt发送请求接受信息，将data信息输出到页面中，注意需要输入自己的KEY，这个项目主要是以js跟css响应式为主，再js文件里我定义了creaChild和generateResponse，creaChild是收到两个参数其中第二个参数是用来判定是人发的信息还是机器人发的，如果是机器人还需要添加smart_toy这个标签然后从chatLi这个变量获取p标签把第一个参数添加到p标签上，generateResponse是发生数据以及KEY请求ChatGPT官网接受信息然后输出到html上。
运用的语法有：document.querySelector，classList.add，JSON.stringify，fetch，innerHTML，trim，toggle
遇到的坑还是再js跟css响应上，太难啦😭，不过还是完成了。（后续会跟新用react来写）。
